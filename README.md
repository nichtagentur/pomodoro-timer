# Pomodoro Timer

A polished, minimal Pomodoro timer built with vanilla JS.

## Features
- Focus / Short Break / Long Break modes
- Start / Pause / Reset
- Auto-cycle: every 4 focus sessions triggers a long break
- Browser notifications + audio beep on completion
- Spacebar to start/pause
- Animated progress bar, color-shifting background, glassmorphism UI
